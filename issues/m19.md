# M19: Intelligent Document Agent

## Overview

**Development Approach:** Backend First
The intelligent chat agent that lives in the input box on all document pages. This agent analyzes user intent, auto-selects tools, chains multiple operations, manages credit limits, and handles session context over 24 hours.

**This is NOT the generation engine (M4).** M4 is middleware that generates documents from text. The agent (M19) is the intelligent layer that:
- Analyzes what the user wants
- Determines which tool(s) to use
- Asks clarification when ambiguous
- Chains multiple tools when needed
- Manages credit limits
- Saves/loads session context
- Rejects non-document tasks

**Difference:**
- **M4 (Generation Engine):** Takes text → returns document (middleware)
- **M19 (Chat Agent):** Takes natural language → routes to correct tool(s) → manages workflow

**Tech Stack:**
- Agent Engine: LLM (DeepSeek API via M4)
- Real-time: Socket.io WebSocket
- Session: MongoDB (messages + embeddings array)
- Streaming: Server-sent events

**Milestone Dependencies:** M1 (Infrastructure), M2 (Frontend Core), M3 (State Management), M4 (Generation Engine)

## Agent Behavior Summary

### Input → Output Flow

```
User Message in Chat Box
    │
    ▼
Intent Analyzer
    │
    ├─── Clear Intent ──► Tool Auto-Selector ──► Execute Tool ──► Output
    │
    └─── Ambiguous Intent ──► Show Options Cards ──► User Selects ──► Execute
```

### Page-Specific Behavior

| Page | Tools Available | Suggests |
|------|----------------|----------|
| Homepage | All 11 tools | - |
| /generate | Generate, Edit, Change Style | - |
| /translate | Translate | Homepage |
| /convert | Convert | Homepage |
| /voice | Voice | Homepage |
| /extract-text | Extract | Homepage |
| /merge-pdf | Merge | Homepage |
| /split-pdf | Split | Homepage |
| /compress-pdf | Compress | Homepage |
| /change-style | Change Style | Homepage |

### Credit Behavior

| Scenario | Action |
|----------|--------|
| Credits ≥75% task | Execute + warn of negative balance |
| Credits <75% + not signed up | Prompt sign up |
| Credits <75% + signed up | Prompt upgrade/buy credits |

### Session Behavior

| Trigger | Action |
|---------|--------|
| User message | Start 24h session |
| 24h inactive | Auto-clear messages + embeddings |
| User enables "Remember me" (later) | Keep embeddings forever |
