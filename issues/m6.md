# M6: Format Templates System

## Overview

**Development Approach:** Backend First
Format template system with presentation pages showing format previews (video/GIF/demo) and detailed AI prompts that generate each format. NO STYLING - pure functionality for format templates.

**Development Notes:**
- No Tailwind CSS - not needed
- No custom UI components with styling - use raw HTML/CSS only when required for testing
- Focus on: Format pages load, preview shows, generation works
- Add styling ONLY when user specifically requests it

**Tech Stack:**
- Frontend: Next.js 14 App Router
- AI: Format prompts (from M4)
- State: Zustand (from M3)

**Milestone Dependencies:** M1 (Infrastructure), M2 (Frontend Core), M3 (State), M4 (Generation Engine), M5 (Generation Page)
