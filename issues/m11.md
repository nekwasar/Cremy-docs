# M11: File Conversion System

## Overview

**Development Approach:** Backend First
File conversion system supporting full matrix of formats. All conversions are COMPLETELY FREE for all users. NO STYLING - pure functionality.

**Development Notes:**
- No Tailwind CSS - not needed
- No custom UI components with styling - use raw HTML/CSS only when required for testing
- Focus on: Upload works, conversion works, download works, preview shows
- Add styling ONLY when user specifically requests it

**Key Rule:** Conversions are COMPLETELY FREE for all users (no credits needed)

**Tech Stack:**
- Frontend: React components
- Conversion: AI-enhanced + libraries
- Export: Puppeteer (PDF), docx.js (DOCX), PptxGenJS (PPTX), SheetJS (XLSX)
- State: Zustand (from M3)

**Milestone Dependencies:** M1 (Infrastructure), M2 (Frontend Core), M3 (State Management), M4 (AI Generation), M7 (Preview)

---

## Supported Conversion Matrix

### Word Processing
- DOC, DOCX, ODT, RTF, TXT ↔ each other + PDF

### Presentations
- PPT, PPTX, ODP ↔ PDF + each other

### Spreadsheets
- XLS, XLSX, ODS, CSV ↔ PDF + each other

### eBooks
- EPUB, MOBI, AZW ↔ PDF

### Markup/Web
- HTML, MD ↔ PDF, DOCX
