# M14: Admin Panel

## Overview

**Development Approach:** Backend First
Admin panel for managing platform settings, pricing, API keys, and viewing analytics. All conversions are COMPLETELY FREE for all users. NO STYLING - pure functionality.

**Development Notes:**
- No Tailwind CSS - not needed
- No custom UI components with styling - use raw HTML/CSS only when required for testing
- Focus on: Data displays, CRUD operations, analytics
- Add styling ONLY when user specifically requests it

**Admin Analytics:** This milestone includes 60 analytics events for deep market research, enabling conversion funnels, cohort analysis, retention tracking, LTV calculation, churn analysis, and revenue tracking.

**Tech Stack:**
- Frontend: React components
- Backend: MongoDB (from M1)
- State: Zustand (from M3)
- Analytics: MongoDB analytics_events collection

**Milestone Dependencies:** M1 (Infrastructure), M2 (Frontend Core), M3 (State Management), M13 (Data Storage)
