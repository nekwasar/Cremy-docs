# M17: SEO Tool Pages

## Overview

**Development Approach:** Frontend First
Additional SEO-optimized landing pages for each tool (variations with different keywords). These are keyword variations for each tool to capture more search traffic. NO STYLING - pure functionality.

**Development Notes:**
- No Tailwind CSS - not needed
- No custom UI components with styling - use raw HTML/CSS only when required for testing
- Focus on: Pages load, SEO metadata, functionality works
- Add styling ONLY when user specifically requests it

**SEO Approach:** Each tool has multiple keyword variations (e.g., "extract text" vs "PDF to text" vs "image to text") to capture different search queries.

**Tech Stack:**
- Frontend: React components
- SEO: Next.js metadata API

**Milestone Dependencies:** M2 (Frontend Core), M11 (File Conversion), M16 (General SEO)
