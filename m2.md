# M2: Frontend Core & Routing

## Overview

**Development Approach:** Backend First
Frontend development with minimal styling for now. Focus on functionality, layout structure, and routing. Add UI components only when needed for testing.

**Tech Stack:**
- Frontend: Next.js 14 with App Router
- UI: Minimal CSS (custom properties), no Tailwind
- State: Zustand for global state
- WebSocket: Socket.io for real-time streaming

**Milestone Dependencies:** M1 (Infrastructure - Docker, Auth, DB, Credit System)

---
